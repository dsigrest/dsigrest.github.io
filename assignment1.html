Introduction

Talk about what you did for the assignment, include pictures of the final product.

Step One: Preparing the Images

Talk about how you picked your images (graphic with distinct outlines), adjusting the threshold in photoshop, using imagetrace and creating outlines in illustrator, as well as cleaning up the extra shapes in illustrator (used the pathfinder tool). I cleaned up the image, In some, I embellished the image to make it morelegible. Then, I exported it as a .dwg- paying careful attention to the scale and units I was exporting it at.

Step Two: Import the File to Rhino

Talk about opening new file in the correct units (inches)- saves time later (learned this the hard way). Check the size, center it. Use the rebuild function to simplify any excess points.

Step Three: Extrude Your Shapes

We create the base for the cookie cutter's walls using the offset tool. I made mine 0.125" thick. Then, I extruded the walls to approximately 2". I created the base by once again offsetting the original cure- this time by 0.5", and extruding a solid surface up about 0.5". Similarly, I created the cookie cutter's inner details by offsetting curves and extruding surfaces.

Step Four: Finishing Up and Exporting

talk about using the boolean union to connect all your shapes, then exporting as an STL.
