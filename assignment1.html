PNW Cookie Cutters

Introduction

As an introduction to Rhino, our class made cookie cutters from shapes we found in images! The goal was to process an image through photoshop and illustrator, import the prepared file into Rhino, then build our cookie cutters from the resulting curves and export them as .STL's. Check out the Rhino files here! Want to see how I did it? Read on!

Step One: Preparing the Images

To make clear cookie cutters, I wanted to choose photos that had distinct silhouettes and graphic shapes. I decided to go with a PNW theme, and chose photos of the space needle, ferns , and sasquatch... Alright, so the sasquatch isn't *really* a graphic shape. But I loved the original photo and wanted to see if I could make something silly work!

First, I imported the images to photshop. I adjusted the threshold to get better control on how image trace would be executed in Illustrator. I cleared the backgrounds from the images and imported them into Illustrator. Using the image trace tool, I produced vector outlines of my images. I spent a couple minutes cleaning up the outlines so I was happy with them. You can do this later in Rhino tool, but I found I could do it with more precision in Illustrator. For the Sasquatch, I added his back leg in by hand, since the original photo did not have enough contrast to capture the leg with the threshhold tool. In a couple caseslines in illustrator, as well as cleaning up the extra shapes in illustrator (used the pathfinder tool). Then, I exported it as a .dwg- paying careful attention to the scale and units I was exporting it at. I chose inches.

Step Two: Import the File to Rhino

Next, I opened up Rhino and chose a template file in inches (the same unit that I used to export the .dwg from Illustrator. I imported my file (again, paying careful attention to units). I checked the size and centered the shape. Using the rebuild function, I simplified any excess points.

Step Three: Extrude Your Shapes

I created the base for the cookie cutter's walls using the offset tool. I made mine 0.125" thick. Then, I extruded the walls to approximately 2". I created the base by once again offsetting the original cure- this time by 0.5", and extruding a solid surface up about 0.5". Similarly, I created the cookie cutter's inner details by offsetting curves and extruding surfaces.
